# 🩺 Skin Disease Detection AI Platform

A full-stack intelligent medical web application for detecting skin diseases using AI.  
The system integrates a **Spring Boot backend**, **Angular frontend**, and a **TensorFlow Lite deep learning model** for image classification.

---

## 🚀 Project Architecture

- **Frontend:** Angular (User Interface)
- **Backend:** Spring Boot (REST API)
- **AI Model:** TensorFlow Lite (Image classification)
- **Database:** MySQL (or your DB if used)

---

## 📌 Features

- Upload skin images for analysis
- AI-powered disease prediction
- RESTful API communication between frontend & backend
- User-friendly Angular dashboard
- Model inference using TensorFlow Lite
- Evaluation and testing scripts

---

## 🧠 AI Component

The AI model is trained to classify skin diseases from images and converted to **TensorFlow Lite** for fast inference in production.
